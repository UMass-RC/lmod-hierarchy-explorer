Unity Module Explorer

For information on using modules, read the documentation on Environment Modules. For help with using modules, ask the Slack helpdesk.

You can click the directories to expand/collapse the listings. You can click the modules to select them. A command will be generated which you can copy/paste into your terminal to load the selected modules. If the selected modules are incompatible or otherwise invalid, the command output window should say so. Some modules are meant for specific types of nodes: cuda modules for GPU nodes, ARM modules for ARM nodes, etc. Since the generated command is run on a non-GPU x86_64 node, there may be warning messages saying that the modules are being loaded on the wrong type of node, but that just means that you should be sure to run your command on the right type of node.
